模块是一个包含 Python 代码的文本文件

使用 import 语句可以将模块导入到其它程序中

Python 中的各个模块提供了自己的命名空间,使用 module.function() 形式调用模块的函数时,要用命名空间名限定函数

# 注释一行
""" ... """ 注释多行


BIF            BIF 的作用
list()         一个工厂函数,创建一个新的空列表
range()        返回一个迭代器,根据需要生成一个指定范围的数字
enumerate()    创建成对数据的一个编号列表,从 0 开始
int()          将一个字符串或另一个数转换为一个整数(若可行)
id()           返回一个 Python 数据对象的惟一标识
next()         返回一个可迭代数据结构(如列表)中的下一项
